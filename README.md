# 💬 WhatsApp Automation using Python

Automate WhatsApp messages using Python — two different methods included.

## 🚀 Features
- **Method 1:** Send WhatsApp messages via Web (PyWhatKit)
- **Method 2:** Send via Android app using ADB (USB Debugging)
- Automatic message scheduling
- Auto send with PyAutoGUI

## 🛠️ Tech Stack
- Python
- PyWhatKit
- PyAutoGUI
- ADB (Android Debug Bridge)

## ⚙️ Requirements
```bash
pip install pywhatkit pyautogui
